🧠 Subconscious

Multi-agent conversation persistence and schema context server, powered by FastMCP on Azure Functions.

MCP Endpoint

/mcp Connect your MCP client (Claude Desktop, Copilot, etc.) to this endpoint.

Browser Views

/view/conversations HTML5/CSS3 conversation browser — client-side Schema.org JSON-LD rendering.
/monitor System health & troubleshooting — Azure Functions, FastMCP, Azure Tables.

Available via MCP

  • Appshow_conversations — JSON-LD list + link to browser
  • Toolcreate_orchestration, list_orchestrations, get_conversation
  • Toolpersist_message, persist_conversation_turn
  • Toollist_schemas, get_schema
  • Toolstore_schema_context, get_schema_context
  • Resourceorchestration://<id>, schema://<name>
  • Promptsummarize_conversation
💬 Conversations ♥ Monitor

All data is exchanged as Schema.org JSON-LD (schema:Action, schema:Conversation, schema:Message) rendered client-side — no React.
